About This Project

This site was authored and developed by Sai Prakash Thota, under the guidance of Dr. Amer. It was created as part of the Curr Topics Human Comp Interac course project.

The aim of this website is to help users understand the essential concepts of staying safe while shopping or banking online. The lessons are built to be simple, interactive, and visually appealing so users can learn comfortably.

The platform includes 10 short tutorials, each covering one important security concept ‚Äî such as strong passwords, phishing attacks, secure website identification, OTP protection, safe vs unsafe Wi-Fi, and secure payment practices.

The interface features a live animated background, glass-effect cards, and a clean layout that works smoothly on both mobile and desktop devices. A built-in progress system marks completed lessons in green to help users track their progress easily.

This project applies key HCI principles such as simplicity, user feedback, visual hierarchy, accessibility, and user-centered design ‚Äî while teaching practical online security skills that are essential in the digital world.
